1. **Recharts tooltip theme hardcoding** — Same pattern in 3 components. A common `chartTooltipStyle` token in `tokens.ts` resolves all at once
2. **Chart data transformation useMemo missing** — 4 chart components recreate arrays on every render
3. **Overlay backdrop-blur overuse** — All 6 overlay components use it. Functional for floating windows but needs consistent policy
4. **Hardcoded hex colors** — `tokens.ts` has `palette` but 5+ files define hex directly. Lint gate doesn't catch inline styles + overlay exclusion scope

- **Mature token system** — `tokens.ts` + `variants.ts` + `index.css` 3-layer, CI lint enforced
- **Consistent focus ring** — All UI primitives have `focus-visible:ring-2`
- **Lazy loading** — All pages use `React.lazy()` code splitting
- **Overlay React.memo** — SuggestionItem, FocusHighlight etc. properly memoized
- **GPU-accelerated animations** — CSS keyframes use only transform/opacity
- **Semantic HTML** — Heading hierarchy, landmarks, real buttons
- **i18n coverage** — Nearly all strings use `t()` function
