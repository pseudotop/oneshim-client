Welcome to ONESHIM

ONESHIM is an AI-powered autonomous office work support agent that monitors your work context and provides intelligent productivity insights.

What will be installed:

  • ONESHIM Desktop Agent — system tray integration
  • Local Web Dashboard (http://localhost:10090)
  • Background context monitoring & AI suggestions
  • Edge image processing (screen capture, delta encoding, OCR)

System Requirements:

  • macOS 10.15 (Catalina) or later
  • Accessibility permissions (for GUI intelligence)
  • Screen Recording permissions (for screen capture)
Privacy: All data stays on your device by default. No data is uploaded to any server without your explicit consent. GDPR Article 17/20 compliant.
