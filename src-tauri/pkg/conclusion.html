Getting Started:

  1. Launch Maekon from your Applications folder or Spotlight search.
  2. Grant Accessibility permissions when prompted (System Settings → Privacy & Security → Accessibility).
  3. Grant Screen Recording permissions for screen capture features (System Settings → Privacy & Security → Screen Recording).
  4. Open the Web Dashboard at http://localhost:10090 to view activity insights and configure settings.
Tip: Maekon runs in the background via the system tray. Right-click the tray icon to access quick settings, pause monitoring, or open the dashboard.
